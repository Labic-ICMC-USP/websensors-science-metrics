# Referência da API

Esta seção é gerada a partir das docstrings presentes em `src/projects/science_metrics`.

Os módulos estão organizados conforme a arquitetura do pipeline:

1. clientes de infraestrutura e fontes de dados;
2. normalização e ingestão;
3. construção e enriquecimento do grafo;
4. resolução de entidades e inferência institucional;
5. steps executáveis pelo WebSensors Flow;
6. configuração e funções utilitárias.
